# 🌿 Frontend Book Study

## 🎯 스터디 목표
자바스크립트와 리액트를 도서 중심으로 학습하며,  
**이론 → 실습 → 프로젝트** 과정을 통해 프론트엔드 개발의 핵심 개념을 탄탄히 익히는 것을 목표로 합니다.

---

## 🗓️ 진행 방식
- **정기 모임:** 매주 수요일 오후 2시  
- **플랫폼:** Discord `dev-leap` 채널  
### 📚 스터디 규칙
1. 매주 한 챕터씩 학습 후 **개인 정리 파일 + 실습 코드 업로드**
2. 스터디 모임 시간에는 **코드 리뷰 및 회고 진행**
3. 리액트 학습 이후에는 **팀 프로젝트를 진행**하여 실무 감각을 높임

---

## 🧩 학습 방식 (제로초의 자바스크립트 입문 기준)
### 📘 개념 부분
- **각자 본인만의 방식**으로 개념을 공부합니다.
(예: 단순히 책을 읽거나, 개인 블로그·깃허브 등에 정리 파일 업로드 등 자유 형식)
### 💻 실습 부분
- **코드 발표 중심 진행**
  - “왜 이렇게 코딩했는지?”, “어떤 부분이 어려웠는지?” 등을 중심으로 자유롭게 발표
  - 서로의 접근 방식을 공유하며 코드 리뷰 진행
